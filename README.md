«Bookly» — это мобильное приложение, предназначенное для описания и учёта прочитанных книг.
Цель проекта — создать удобное цифровое пространство, где пользователь может хранить информацию о своих прочитанных произведениях, формировать подборки и отмечать любимые книги.
Приложение предоставляет структурированный и функциональный инструмент для систематизации прочитанного материала, ведения кратких пересказов и анализа личных читательских достижений.

Функциональные возможности:
- Добавление книг: пользователь может вносить книги в приложение с указанием обложки, названия, автора, жанра и краткого описания прочитанного произведения.
- Создание подборок: возможность формировать подборки книг из уже добавленных произведений с указанием обложки и описания подборки.
- Учёт прочитанного: приложение отображает количество добавленных книг и созданных подборок, что позволяет отслеживать объём прочитанного материала.
- Вкладка «Избранное»: отдельный раздел для сохранения любимых или наиболее значимых книг, чтобы быстро получить к ним доступ.
- Поиск книг: реализован удобный поиск по названию, автору и жанру, что позволяет быстро находить нужное произведение в коллекции.
